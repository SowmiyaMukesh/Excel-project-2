# HR Analytics Dashboard (Excel)

# Project Overview

In this project, I worked with an HR dataset and built an interactive dashboard in Excel to analyze employee data and identify patterns related to workforce distribution and attrition.
The main goal was to take a raw dataset, clean and structure it properly, and then convert it into a visual dashboard that is easy to understand and explore.
________________________________________
# Dataset

The dataset includes employee-related information such as:
•	Employee demographics
•	Department and job roles
•	Salary and experience details
•	Attrition status
________________________________________
# Data Preparation

Before building the dashboard, I worked on preparing the dataset for analysis:
•	Cleaned and structured the data
•	Handled inconsistencies in values
•	Organized the dataset for analysis using Excel features
________________________________________
# Tools & Techniques Used

•	Microsoft Excel
•	Data Cleaning
•	Pivot Tables
•	Pivot Charts
•	Slicers
•	Dashboard Design
________________________________________
# Dashboard Development

I used pivot tables and pivot charts to summarize the data and built a dashboard to present key employee insights.
The dashboard allows users to:
•	View employee distribution across different categories
•	Analyze attrition trends
•	Interact with the data using slicers
________________________________________
# Key Insights

•	Employee distribution varies across departments and roles
•	Attrition patterns can be observed across different segments
•	The dashboard helps in quickly understanding workforce trends
________________________________________
# Conclusion

This project helped me understand how to transform raw HR data into a structured and interactive dashboard using Excel. It improved my skills in data cleaning, analysis, and visualization.
